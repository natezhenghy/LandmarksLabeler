# LandmarksLabeler

## 数据格式
 - ParentFolder
     - img
         - imgfolder1
             - 1.jpg
             - 2.jpg
         - imgfolder2
     - fpp
         - imgfolder1
             - 1.txt
             - 2.txt
         - imgfolder2
## 使用方法
1. 打开数据库文件夹
2. 鼠标拖动调整landmarks
3. 单击保存更新landmarks(txt)文件
4. 使用方向键浏览前/后一张图片/文件夹

## 功能
- [x] 浏览
- [x] 修改landmarks
- [x] 保存
- [ ] 记录数据库异常情况的Log
- [ ] 记录操作时间
- [ ] 选中某一landmark时, 提供标准缩略图作为提示
- [ ] 优化拖动landmark时的UI提示(e.g.鼠标指针变化, landmark颜色变化)
